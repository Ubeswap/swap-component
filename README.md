# Ubeswap Swap Component

Ubeswap-Swap allows you to add swap module of Ubeswap to your app with ease!

## Installation

```
$ npm install --save ubeswap-swap-dev
$ yarn add ubeswap-swap-dev
```

## Usage

### Normal Usage

```jsx
  import Swap from 'ubeswap-swap-dev';
  
  function App(){
    return (
      <Swap />
    );
  }
```

### Dark Mode Usage

```jsx
<Swap useDarkMode/>
```
